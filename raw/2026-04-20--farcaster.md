# Farcaster

## One-paragraph

Decentralized social **protocol** on Ethereum (contracts on [[Base]]-adjacent infrastructure) originally built by **Dan Romero** and **Varun Srinivasan** at **Merkle Manufactory**. Uses a permissioned registry contract for identity (FID) and off-chain Hubs for message storage/replication, with multiple independent clients (Warpcast being the canonical one, later renamed Farcaster App). Known for the **Frames** / **Mini Apps** primitives that made in-feed interactive content a native pattern. Ownership transferred to **Neynar** in Q1 2026 for ~$1B; Romero & Srinivasan pivoted to a separate Farcaster-based wallet app.

## Key facts

- **Founders:** Dan Romero (@dwr), Varun Srinivasan (@v)
- **Founding company:** Merkle Manufactory
- **Launch (public, paid signups):** August 2023
- **Chain:** Ethereum / Base (OP Mainnet for the registry contract historically)
- **2026 stats (December 2025 reported):** 250,000 MAU, 100,000+ funded wallets
- **Acquisition:** **Neynar acquired Farcaster on 2026-01-22 for ~$1B** — transferred protocol contracts, code repos, Farcaster App (née Warpcast), and Clanker
- **Merkle wind-down:** refunded ~$180M to investors during the transition
- **Founders' next bet:** Farcaster-based wallet app (separate from Neynar's direction)

## Protocol primitives

- **FID** (Farcaster ID) — on-chain identity, minted for a small fee
- **Casts** — posts (short-form messages, can include media, channels, mentions)
- **Channels** — topical subgraphs (like subreddits or Discord channels)
- **Hubs** — nodes that replicate the protocol state; anyone can run one
- **Storage units** — users pay for storage (rented, not perpetual), creates protocol economic base
- **Frames** — interactive in-feed content (polls, mints, games) — launched Jan 2024
- **Mini Apps** (2026 launch) — expanded Frames into richer interactive apps in-feed

## Clients

- **Warpcast** → renamed **Farcaster App** (Merkle-built, now Neynar-owned)
- **Clanker** (memecoin launcher on Farcaster) — transferred to Neynar
- **Neynar's infrastructure** — most Frames / Mini Apps development uses Neynar as the provider
- **Others historically:** Supercast, Yup, various Farcaster clients (less canonical post-Warpcast dominance)

## Token / economics

- **No native token** at the protocol level
- Economic surface is **storage rental** (FIDs, storage units) and **Frames economics** (builders monetize interactions)
- **DEGEN** is a Farcaster-culture token (not protocol-native) that became the de facto tipping currency — see [[DEGEN]]

## Why this matters for the wiki

- **The default "social protocol on Ethereum"** — anchor for the protocol layer vs app layer distinction
- **Frames / Mini Apps** invented a new interaction primitive that's being copied everywhere (Zora's feed, Twitter/X's frame-like embeds, Telegram bots converging)
- **Ownership transition** (Merkle → Neynar, 2026) is a major event — sets a pattern for how "social protocols" get consolidated
