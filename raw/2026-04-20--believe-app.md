# Believe (believe.app)

## User thesis (Alex)

> X.com @ mention as a way to get virality.

The core insight: **the tweet IS the launch surface**. Launching a token requires posting publicly on X, which broadcasts the token to the creator's network in the same act. The launch and the virality loop are the same action — not "launch, then promote" but "promote-by-launching."

## Research summary

Solana-based memecoin launchpad. Users tag **@launchacoin** on X with a ticker name; Believe's backend deploys an SPL token with the corresponding ticker and replies with a token link.

Founded by **Ben Pasternak**, the same person who ran [[Clout (clout.me)]] — this is that project's evolution.

## Mechanism

- **Launch flow:** tweet mentioning @launchacoin + ticker → backend SPL deploy → reply confirms with link
- **Pricing:** dynamic bonding curve; price rises as buyers purchase
- **Graduation:** at $100k market cap, token migrates off the curve into **Meteora** (Solana liquidity protocol) for deep trading
- **Fees:** 2% per trade
  - 1.0% → token creator
  - 0.1% → "scouts" (referrers)
  - 0.9% → Believe platform
- **Creator alignment:** if creator links their X account, they receive half of all fees; creators also claim 5% of supply
- **Platform token:** $LAUNCHCOIN (rebranded from $PASTERNAK in early May 2025); reached $200M+ market cap within days

## Chain

Solana.

## Notable features

- X-native launch flow — no separate web UI required to mint
- Automatic migration to Meteora on graduation (vs pump.fun's Raydium migration)
- Creator/scout/platform three-way fee split rewards distribution not just trading
